 ## <h2 id="1">一、HKC是什么</h2>
 ### 1.1 HKC是什么
  * HKC是一种以人为中心的知识分类方法。通过识别人与外物的关系，即人与自我， 人与神，人与人，人与自然的关系，来定义一种新的知识分类。

 ### 1.2 解决的问题
  * 在前人图书分类基础上，重新定义/优化知识分类方法
  * 解决在信息大爆炸时代信息碎片化问题

 ### 1.4 适用范围
  * 人类知识管理分类
  * 领域知识管理分类
  * 个人知识管理分类

 ### 1.5 口号
  * 没有哲学上的思考，就没有行动上的的指导。
  * No thinking in philosophy, No direction in action.
